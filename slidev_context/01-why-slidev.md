# Why Slidev?

## Motivation

Slidev aims to provide a **flexible, developer-friendly presentation tool** that focuses on content and leverages web technologies. Unlike traditional slide-making tools like Microsoft PowerPoint and Apple Keynote, Slidev is designed specifically for developers.

## Key Motivations

1. **Developer Focus**: Help developers create more interesting and expressive presentations
2. **Familiar Technologies**: Use technologies developers already know
3. **Content-Focused**: Enable content-focused slide creation
4. **Maximum Flexibility**: Allow maximum flexibility and interactivity

## Core Features

### 1. Markdown-based
Organize slides in a single plain text file using Markdown syntax.

### 2. Developer-friendly
First-class support for code snippets with syntax highlighting.

### 3. Highly Themable
Themes can be shared and distributed via npm packages.

### 4. Web-based
"**Nothing is impossible**" - anything possible in a web app can be applied to slides.
- Use Vue components
- Add interactivity
- Leverage modern web APIs

### 5. Interactive
Can use Vue components and interact with slides during presentation.

### 6. Portable
Export to multiple formats:
- PDF
- PPTX
- PNGs
- Single-page application (SPA)

## Unique Advantages Over Competitors

Compared to other presentation tools:

- **More concise** than Reveal.js
- **Supports Vue components** natively
- **Built on Vite** for instant hot module replacement (HMR)
- **Supports UnoCSS** out of the box for rapid styling
- **Open source** with strong community support

## Fundamental Philosophy

> "Slidev aims to provide flexibility and interactivity for developers to make their presentations much more interesting, expressive, and attractive."

The tool is built by developers, for developers, embracing modern web development practices and tools.
